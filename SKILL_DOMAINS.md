# Magic Skills - 6 个专业领域技能清单

本文档详细定义了 Magic Skills MVP 阶段聚焦的 6 个专业领域的技能需求。

---

## 领域 1：Java 后端开发 AI 编程技能集

**目标用户**：Java 后端开发工程师、架构师、技术负责人

**核心技术栈**：Spring Boot、MyBatis/Hibernate、MySQL/Redis、Maven/Gradle、JUnit、Docker/Kubernetes

### 1.1 代码生成类技能

#### spring-boot-controller-gen
- **描述**：根据业务需求生成 Spring Boot Controller 代码
- **输入**：业务描述、API 路径、请求/响应模型
- **输出**：完整的 Controller 类（包含 REST 注解、依赖注入、异常处理）
- **场景**：快速搭建 RESTful API 接口

#### spring-boot-service-gen
- **描述**：生成 Spring Boot Service 层代码
- **输入**：业务逻辑描述、DAO 接口、事务要求
- **输出**：Service 接口和实现类
- **场景**：实现业务逻辑层

#### spring-boot-dao-gen
- **描述**：生成 DAO/Repository 层代码
- **输入**：实体类、数据库操作需求
- **输出**：DAO 接口和 MyBatis Mapper XML
- **场景**：数据访问层开发

#### jpa-entity-gen
- **描述**：生成 JPA Entity 实体类
- **输入**：数据库表结构、关系映射需求
- **输出**：JPA Entity 类（包含注解、关系映射）
- **场景**：数据库实体建模

#### rest-api-doc-gen
- **描述**：生成 REST API 文档（Swagger/OpenAPI）
- **输入**：Controller 代码、业务描述
- **输出**：Swagger 注解、API 文档
- **场景**：API 文档自动化

### 1.2 代码分析类技能

#### code-review-java
- **描述**：Java 代码审查
- **输入**：Java 源代码
- **输出**：代码质量问题列表、改进建议
- **场景**：代码审查、质量提升

#### performance-analyzer
- **描述**：性能分析
- **输入**：代码、性能指标要求
- **输出**：性能瓶颈分析、优化建议
- **场景**：性能优化

#### security-checker
- **描述**：安全漏洞扫描
- **输入**：源代码
- **输出**：安全漏洞列表、修复建议
- **场景**：安全审计

#### code-complexity
- **描述**：代码复杂度分析
- **输入**：源代码
- **输出**：圈复杂度、代码行数、维护性指数
- **场景**：代码质量评估

#### dependency-analyzer
- **描述**：依赖关系分析
- **输入**：项目结构、pom.xml/build.gradle
- **输出**：依赖关系图、冲突检测
- **场景**：依赖管理

#### spring-best-practices
- **描述**：Spring 最佳实践检查
- **输入**：Spring 项目代码
- **输出**：最佳实践违背列表、改进建议
- **场景**：代码规范化

### 1.3 配置生成类技能

#### spring-config-gen
- **描述**：生成 Spring 配置文件
- **输入**：功能需求、环境配置
- **输出**：@Configuration 类或 XML 配置
- **场景**：Spring 配置管理

#### docker-compose-gen
- **描述**：生成 Docker Compose 文件
- **输入**：应用架构、服务依赖
- **输出**：docker-compose.yml
- **场景**：容器化部署

#### k8s-deployment-gen
- **描述**：生成 Kubernetes 部署文件
- **输入**：应用配置、副本数、资源限制
- **输出**：Deployment、Service、Ingress YAML
- **场景**：K8s 部署

#### application-yml-gen
- **描述**：生成 application.yml 配置文件
- **输入**：环境配置、数据库连接、第三方服务
- **输出**：application.yml
- **场景**：应用配置管理

#### logback-config-gen
- **描述**：生成 Logback 日志配置
- **输入**：日志级别、输出目标、滚动策略
- **输出**：logback.xml
- **场景**：日志管理

### 1.4 测试类技能

#### junit-test-gen
- **描述**：生成 JUnit 单元测试
- **输入**：被测类、测试场景描述

- **输入**：API 接口、测试数据
- **输出**：集成测试代码
- **场景**：端到端验证

#### api-test-gen
- **描述**：生成 API 测试
- **输入**：API 文档、测试用例
- **输出**：API 测试代码（RestAssured 等）
- **场景**：API 验证

#### performance-test-plan
- **描述**：生成性能测试计划
- **输入**：API 接口、并发数、持续时间
- **输出**：JMeter/Gatling 测试脚本
- **场景**：性能测试

### 1.5 文档类技能

#### code-comment-gen
- **描述**：生成代码注释
- **输入**：源代码
- **输出**：包含 JavaDoc 的代码
- **场景**：代码文档化

#### api-documentation
- **描述**：生成 API 文档
- **输入**：Controller 代码、业务描述
- **输出**：Markdown/API 文档
- **场景**：API 文档编写

#### architecture-doc
- **描述**：生成架构文档

- **输出**：架构设计文档
- **场景**：技术文档

#### deployment-guide
- **描述**：生成部署指南
- **输入**：部署环境、步骤
- **输出**：部署文档
- **场景**：运维文档
